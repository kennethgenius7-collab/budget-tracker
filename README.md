# Personal Budget Tracker

A simple, elegant personal budget tracker that runs entirely in your browser with local storage.

## Features
- Track income and expenses
- Categorize transactions
- Visual charts for spending analysis
- Monthly summary
- Filter transactions by type and category
- Local storage persistence
- Responsive design

## How to Use
1. Download all files to a folder
2. Open `index.html` in your browser
3. Start adding transactions!

## File Structure
- `index.html` - Main HTML file
- `style.css` - Styling
- `script.js` - JavaScript functionality

## Technologies Used
- HTML5
- CSS3 (with CSS Grid & Flexbox)
- Vanilla JavaScript
- Chart.js for visualizations
- Font Awesome icons
- Local Storage API

## Customization
- Modify color scheme in CSS `:root` variables
- Add new categories in the JavaScript category mapping
- Change currency format in `formatCurrency()` method

## Browser Compatibility
Works on all modern browsers (Chrome, Firefox, Safari, Edge)
